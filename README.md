I created this leave tracker with the intent of allowing myself to easily visualise the existing manpower in my company at any point in time, 
as well as to track the number of leaves/offs that my men had. 

In this tracker, my men are allowed to personally submit requests for leaves/offs, which would then submit a request to the admins for
approval/rejection. If approved, these requests would then be reflected on a calendar from the admin panel.
